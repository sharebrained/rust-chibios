# chibios

[Rust] FFI wrapper around [ChibiOS] 2.6. This wrapper was created for the [PortaPack]
project.

[Rust]: https://www.rust-lang.org/
[ChibiOS]: http://www.chibios.org/
[PortaPack]: http://www.sharebrained.com/portapack/
